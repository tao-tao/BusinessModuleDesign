コマンドラインからHTMLを生成

htmlgen.jarを使用するとAmaterasERDで作成した*.erdファイルからコマンドラインでHTMLドキュメントを生成することができます。 htmlgen.jarは以下のURLからダウンロードできます。
http://sourceforge.jp/projects/amateras/releases/?package_id=11865

htmlgen.jarは以下のようにして実行します。 

$ java -jar htmlgen.jar erdファイル 出力先ディレクトリ

出力先ディレクトリは予め作成しておく必要があります。
